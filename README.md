Chocolate Sales Analytics Dashboard

An interactive Power BI dashboard analyzing chocolate product sales performance across regions, sales representatives, and product categories.

Project Overview

This dashboard analyzes chocolate sales data across multiple dimensions — sales representatives, product types, regions, and time periods. It provides clear visibility into performance trends and helps identify growth opportunities.

Features
Sales by Product — Identifies best-selling chocolate products
Sales Rep Performance — Tracks individual and team performance
Regional Breakdown — Compares sales across regions
KPI Tracking — Monitors Total Sales, Units Sold, and Revenue
Dynamic Filters — Interactive slicers for product, region, date, and sales rep
DAX Measures — Custom calculations for performance tracking
Tools & Technologies
Power BI Desktop — Dashboard development and visualization
DAX — Custom measures and KPIs
Excel / CSV — Data source
